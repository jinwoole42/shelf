# Bit operators

정수는 이진법으로 표현될 수 있고, 그래서 정수만 비트연산이 가능하다.  
char a = 5, a = 0000 0101  
char b = 1, b = 0000 0001  
컴퓨터에겐 이쪽이 훨씬 쉽다.

### 1. and
a & b -> 같은 자리의 비트를 비교해 둘 다 1이라면 1, 아니면 0 반환
### 2. or
a | b -> 하나라도 1이면 1 반환
### 3. xor
a ^ b -> 두 비트가 같으면 1, 다르면 0 반환
### 4. not
~a -> 모든 비트 반전
### 5. left, right shift
a << 3, b >> 1 -> 비트를 전부 왼/오른쪽으로 n만큼 이동
### 6. 축약연산
a &= b -> += 와 비슷하다. 축약한 결과를 다시 a에 집어넣는다.
